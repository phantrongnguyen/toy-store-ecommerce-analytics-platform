About the Project

End-to-end e-commerce analytics platform powered by data science

📊

Toy Store E-Commerce Analytics Platform

A complete data pipeline from raw CSV data to machine learning predictions. This platform ingests e-commerce session data into a SQL Server OLTP database, transforms it through SSIS ETL into a star-schema data warehouse, and serves predictions via a FastAPI backend with an interactive dark-mode dashboard.

Mission

Democratize e-commerce revenue prediction with accessible, accurate machine learning tools for small to medium toy retailers.

🔧

Tech Stack

Python, FastAPI, Streamlit, SQL Server, SSIS, Scikit-learn, Power BI, Chart.js — full-stack data engineering and analytics.

📈

Impact

Random Forest model achieves 95.9% R² score, enabling data-driven inventory and marketing decisions for e-commerce stores.

🔮

Future

Planned enhancements include purchase prediction, refund prediction, real-time streaming, and dynamic dashboard updates.
